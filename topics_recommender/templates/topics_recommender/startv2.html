{% translate "Background" %}

{% blocktranslate %} The Information and Computer Engineering department is planning a new course about open-source software (OSS). In the course, students will learn about OSS and contribute to real OSS projects on the Internet. Since OSS contributions are essentially a kind of voluntary work, the school would like to collect a list of topics that students would be willing to work on. This tool allows you to browse the available project topics and choose some for your theme as if you were a participant in the new course. Each topic is like a special tags that shows a project's purpose, subject area, community, or programming language. They are used to classify source code repositories and make it easier for people to find projects that interest them. {% endblocktranslate %}

{% translate "Instructions" %}

{% blocktranslate %} For this task, please do the following: {% endblocktranslate %}

  1. {% translate "Use the Search page to explore topics that might interest you." %}
  2. {% translate "Look at the recommended topics and read their descriptions." %}
  3. {% translate "If one of the recommended topics sounds interesting, click Search this topic." %}
  4. {% translate "When you find a topic you would choose, click Choose this topic." %}
  5. {% translate "Continue searching topics until you have chosen a total of five topics." %}

{% blocktranslate %} The Search page will show a text input for you to type the name of a topic. An autocomplete menu will show you the list of matching topics as you type. If the topic does not exist, it will not appear in the autocomplete box. In that case, please check your spelling or try a different topic. Click the Submit button to see details about a topic and a list of other recommended topics. {% endblocktranslate %}

{% blocktranslate %} When searching a topic, a Choose this topic button will appear under the topic description. Click that button to add the topic to your list of chosen topics at the top of the page. When the Chosen topics list is full, a Submit button will appear below the list. If you want to unselect a topic, click the [ Delete ] link next to it in the Chosen Topics list. {% endblocktranslate %}

{% blocktranslate %} The search results will show a list of topic names and their descriptions. When the page first loads, the descriptions are hidden for all topics except the searched topic. Click on the name of a topic to show or hide its description. Below the description of each topic is a Search this topic link. Click that link to search the specified topic. {% endblocktranslate %}

{% blocktranslate %} While searching for topics, you can explore other topics that do not appear in the search results by clicking the New Search button under the searched topic description. {% endblocktranslate %}

{% blocktranslate %} Please click the button below when you are ready to begin. {% endblocktranslate %}
