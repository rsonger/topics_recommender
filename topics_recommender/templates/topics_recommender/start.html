{% translate "Background" %}

{% blocktranslate %} KIT is planning a new course for the Information and Computer Engineering curriculum about open-source software (OSS). In the course, students will learn about OSS and contribute to real OSS projects on the Internet. Since OSS contributions are essentially a kind of voluntary work, the school would like to collect a list of topics that students would be willing to work on. This tool allows you to browse the available project topics and choose one for your theme as if you were a participant in KIT's new course. Each topics is like a special tags that shows a project's purpose, subject area, community, or programming language. They are used to classify source code repositories and make it easier for people to find projects that interest them. {% endblocktranslate %}

{% translate "Instructions" %}

{% blocktranslate %} For this task, please do the following: {% endblocktranslate %}

  1. {% translate "Use the Search page to explore topics that might interest you." %}
  2. {% translate "Look at the recommended topics and read their descriptions." %}
  3. {% translate "If one of the recommended topics sounds interesting, click Search this topic." %}
  4. {% translate "When you find a topic you would choose, click Choose this topic to finish searching." %}

{% blocktranslate %} The Search page will show a text input for you to type the name of a topic. An autocomplete menu will show you the list of matching topics as you type. If the topic does not exist, it will not appear in the autocomplete box. In that case, please check your spelling or try a different topic. Click the Submit button to see details about a topic and a list of other recommended topics. {% endblocktranslate %}

{% blocktranslate %} The search results will show a list of topic names and their descriptions. When the page first loads, the descriptions are hidden for all topics except the searched topic. Click on the name of a topic to show or hide its description. Below the description of each topic is a Search this topic link. Click that link to search for the specified topic. {% endblocktranslate %}

{% blocktranslate %} If you want to explore another topic that does not appear in the search results, click the New Search button at the top of the page. {% endblocktranslate %}
